# ATM Management System (Python)

## Overview

This is a simple ATM Management System developed in Python using object-oriented programming concepts. The project simulates basic banking operations such as balance inquiry, cash withdrawal, and cash deposit through a console-based interface.

## Features

* Account login using a PIN
* Balance inquiry
* Cash withdrawal
* Cash deposit
* Transaction verification
* Transaction receipt generation
* Random transaction number generation
* Menu-driven user interface

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* Random Module

## Concepts Implemented

* Classes and Objects
* Constructors
* Methods and Attributes
* Loops and Conditional Statements
* User Input Handling
* Basic Banking Operations

## How to Run

1. Install Python on your system.
2. Save the source code as `atm_system.py`.
3. Open a terminal or command prompt.
4. Run the program using:

```bash
python atm_system.py
```
